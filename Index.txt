Project #4: 

	- Name: Bluetooth Remote Control Car

	- Purpose: To be able to gain competency in using Bluetooth modules and just be able to put something together that's cool

	- Components: 2 Arduino Uno's, 2 USB cords to connect Arduino to PC, 1 potentiometers, 2 buttons, 5v external battery power source, normal 5v micro servo motor, 4 	  mini rubber wheels, hc05 Bluetooth module, bread board, jumper wires, micro RGB light, 3 220k ohm resistors.

	- No Libraries used for source code
 
	- CAD Platform: Onshape.com (free)
